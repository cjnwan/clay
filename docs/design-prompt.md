# Claude Design 提示词：《捏黏土》APP 设计风格重塑

（直接整段粘贴使用；如工具支持看网页，附上线上地址效果更好）

---

为儿童 3D 黏土创作游戏《捏黏土》重新设计一套完整的移动 APP 视觉设计系统与关键界面。
现有网页版可直接体验：https://cjnwan.github.io/clay/ （请先玩一遍再动手设计）。

## 产品背景

- 用户：4–9 岁儿童（很多不识字）+ 陪玩家长；手机竖屏优先，平板次之；触摸操作，胖手指。
- 产品是一个"数字黏土工坊"，双空间结构：
  1. **🧸 手办工坊（主页）**：无重力转盘台，一块一块捏黏土再组装成手办。
     - 底部面板 4 行：块架 chips 行（每块泥的实时 3D 缩略图 + ＋新块）→ 12 色黏土色板 →
       内容行（随 tab 切换）→ 控制行（⬅回黏土板 ┃形状┃雕刻┃装扮┃分段控件 ↻转一转 ✓完成）。
     - **形状 tab（蓝）**：10 种基础形（3 个 Q 版身体 + 球/蛋/扁饼/香肠/方砖/圆锥/半球）+ 拉高/压扁按钮；双指捏合调整块大小。
     - **雕刻 tab（橙）**：戳坑/鼓包二选一、四档笔刷、撤销；点一下戳、按住划出沟，沟在手指下实时成形。
     - **装扮 tab（粉）**：14 种部件货架（招牌是"玩偶服头套"——穿上后开脸露出里面的动物脸；
       还有眼睛/嘴/圆点/耳朵/小手/色片/轮子/翅膀/尖角/尾巴/小帽/星星/爱心），拖到身上按下去，
       成对部件自动镜像；此 tab 色板管部件颜色。
     - **组装**：把块架上的 chip 拖到转盘上的块身上装上去（第二指捏合调大小、旋转调方向），
       拆下来永远回块架不消失；✓ 完成把整个组装体送到黏土板变成玩具。
  2. **黏土板（玩耍区）**：物理沙盒。点色板掉黏土球、慢速靠近自动黏住、摆稳自动"定型"、
     点一下跳、双击拆开、剪刀/画笔/捏一捏工具、🎬 自动表演教学、🎥 拍环绕短片、
     拖出盘外收走、手办们会轻轻呼吸和眨眼。
- 存档即分享：网址就是分享链接；作品跨设备打开。

## 现有视觉语言（这是要"升级"而非"推翻"的魂）

- **暖陶土世界**：米色背景 #EFE0C8，木质台面棕色系，无冷灰、无纯黑白 UI。
- **12 色黏土色板**：#E0584B 红 / #F2A34D 橙 / #F6D155 黄 / #7DBB5F 绿 / #53A7DD 蓝 /
  #A06BC9 紫 / #F7F3EC 白 / #3A3430 黑 / #9C6B4A 棕 / #F2A7C4 粉 / #A9A9AE 灰 / #F2E0C2 奶油。
- **手绘感图标**：全套内联 SVG，歪斜描边、暖色填充、每个按钮带 ±2–4° 随机微旋转；
  色板按钮是不规则"黏土团"圆角；按钮是白底圆形凸起 + 内阴影。
- **模式 = 主题色**：形状蓝 / 雕刻橙 / 装扮粉，选中 tab 时内容行背景与画布指示物同步染色
  （不识字儿童靠颜色感知当前模式，这条必须保留）。
- **质感层**：清漆黏土材质、脚下软阴影、一切落定动作都有"挤压回弹"、眼睛随机眨、
  手办呼吸、咕叽/啵/咚拟音 + 手机震动。

## 设计任务（交付物）

1. **完整设计系统**：
   - 色彩 tokens（背景/表面/主题三色/语义色 + 12 黏土色的精确规范与使用规则）；
   - 字体（中文圆体优先；"文字只是辅助"原则——所有关键操作不依赖文字）；
   - 图标体系：重绘全套约 40 个图标（工具、形状、部件、控制），统一笔触语言，给出网格与描边规范；
   - 组件库：圆形工具按钮、分段 tab 控件、块架 chip（含实时缩略图位）、色板团、提示条、
     ✓/↺ 确认气泡、长按进度反馈、上限抖动反馈；
   - 布局：竖屏栅格、底部操作区高度、安全区、横滑行规范；
   - 动效规范：回弹曲线（现用 easeOutBack/spring）、时长档位、转场（板⇄工坊的相机飞行）；
   - 声音与触觉建议（现有拟音风格：湿黏、低频、有机抖动）。
2. **关键界面 8 张**：工坊-形状 / 工坊-雕刻 / 工坊-装扮 / 块架组装中（chip 拖拽幽灵态）/
   黏土板玩耍 / ✓ 完成落板瞬间 / 作品分享卡（含二维码位）/ 启动页。
3. **状态设计**：空转盘引导、块数/部件上限、长按删除、误操作撤销、离线。

## 硬约束

- 4 岁不识字可独立使用：图标自明、≥44pt 触达、颜色即模式、无失败态（一切可撤销/可拿回）；
- 游戏 3D 画布是主角：UI 贴边、半透明化、绝不遮挡转盘中心；
- 单手竖屏优先；内容按钮超过 8 个横向滑动，不加行；
- 保持温暖手作感：允许暗色模式但必须仍是"温暖的黏土工坊夜晚"，不是冷黑。

## 风格方向

在"手工黏土工作室"的基础上做**品质升级**：参考 Toca Boca 的大胆用色与无文字交互、
任天堂式的动效手感、阿德曼（Aardman）定格动画的黏土材质真实感。
避免：塑料感 Material Design、成人工具感、赛博/科技风、锐利几何。

## 输出格式

给出设计 tokens 表（色值/圆角/间距/字号/动效时长）+ 每张界面的标注说明
（组件复用关系、点击目标尺寸、模式色应用位置），确保工程可以照稿实现。
